# JS Backend Fundamentals

Coleção de exemplos práticos dos **fundamentos de JavaScript** para backend, organizados por tópico.

## Tópicos

| #  | Arquivo              | Conceito           |
|----|----------------------|--------------------|
| 01 | `variables.js`       | var, let, const    |
| 02 | `strings.js`         | Strings e template literals |
| 03 | `numbers.js`         | Tipo Number        |
| 04 | `booleans.js`        | Tipo Boolean       |
| 05 | `truthy-falsy.js`    | Truthy e Falsy     |
| 06 | `dynamic-typing.js`  | Tipagem dinâmica   |
| 07 | `conversions.js`     | Conversão de tipos |

## Stack

- Node.js 18+
- Node.js Test Runner (built-in)

## Como executar

```bash
# Executar um exemplo
node src/01-variables.js

# Executar testes
npm test
```
